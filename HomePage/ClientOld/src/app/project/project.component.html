Guild Quest

Based on a project that I worked on while attending The Software Guild. The original project was developed using .Net 4.5.2 and Bootstrap 3.

The current project is using .Net 4.7.1 and Bootstrap 4. Since the default MVC template uses Bootstrap 3, I had to convert it to Bootstrap 4.

This project is still a work in progress and has not been optimized for mobile platforms.
It is best viewed from a laptop or desktop.

Features

ASP.Net MVC
ASP.Net API
Entity Framework 6.2
Bootstrap 4.0
HTML5/CSS3/jQuery
SQL Server 2016
Linq
Twilio - used for two party authenication
Microsoft Identity
CKEditor - inline text editor
